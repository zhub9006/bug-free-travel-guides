## 🤝 How to Contribute

### Want to add a destination?
1. Read the [Submission Template](guides/submission-template.md)
2. Fill in all sections with verified data + 2+ sources
3. Open a PR with the `destination-request` label
4. Wait for community review
5. Celebrate when it gets merged! 🎉

### Found outdated info?
1. Open an issue with the `research-update` label
2. Describe what changed and provide 2+ sources
3. A moderator will review and update

### Not sure where to start?
1. Check the [Community Roadmap](guides/community-roadmap.md) for priority tasks
2. Look for issues labeled `good first issue` or `quick win`
3. Even a typo fix helps! Every contribution counts.
