# Contributing — Bug-Free Travel Guides 🪲🚫

Thank you for wanting to contribute! This guide explains how to add a new destination, update an existing guide, or report a bug sighting. Every contribution helps us build the world's definitive bug-free travel resource.

---

## 🔍 How It Works

1. **Pick a destination** — Open a [GitHub Issue](https://github.com/zhub9006/bug-free-travel-guides/issues/new?labels=destination-request) to claim it or check the [Community Roadmap](guides/community-roadmap.md) for un-researched destinations.
2. **Research** — Use the [Submission Template](guides/submission-template.md) to structure your guide. Every destination must be cross-referenced with at least **2 independent sources**.
3. **Write** — Follow the naming convention: `guides/N-destination-name.md` (where N is the next available number).
4. **Submit** — Open a Pull Request! Community reviewers will verify your work.
5. **Publish** — Once approved, your guide is merged and added to the README index.

## 📋 Contribution Standards

### Every Destination Guide Must Include:

| Field | Required? | Notes |
|---|---|---|
| Destination name + country | ✅ | Include country/region |
| Bug safety score (0–10) | ✅ | See the legend in any guide |
| Why it's safe (ecological reasons) | ✅ | At least 2 different factors |
| Safest seasons to visit | ✅ | Month-by-month calendar |
| Budget level | ✅ | $–$$$$ |
| Top activities | ✅ | 3–5 destinations |
| Bug breakdown by type | ✅ | Mosquitoes, ticks, sandflies, etc. |
| Warnings & traps | ✅ | Seasonal traps, places to AVOID |
| Source citations (2+) | ✅ | At least 2 independent sources |
| Pro tips for phobic travelers | ✅ | 3 actionable tips |
| Personal visit dates (if applicable) | ⭐ Optional | Helps verify current conditions |

---

## 📏 Verification Protocol

All submitted guides must follow this verification protocol:

1. **Source Collection** — Minimum 2 independent sources
2. **Cross-Validation** — Check contradictory evidence
3. **Seasonal Analysis** — Provide month-by-month data
4. **Community Review** — Core contributors verify accuracy
5. **Ongoing Monitoring** — Check for climate changes every 12 months

### Quality Checks Before Publishing:

- [ ] All bug scores are specific and justified
- [ ] Seasonal timing is accurate and detailed (not vague)
- [ ] At least 2 sources are cited with dates
- [ ] Warnings and seasonal traps are clearly marked
- [ ] No unverified claims (no "I heard it's safe" without a source)

## 🚫 Bug Reports & Climate Anomalies

If you visit a "bug-free" destination and encounter unexpected insects:

1. Open a GitHub Issue with label `bug-report`
2. Include: destination, location, date, what you saw, and photo if possible
3. This helps us update our guides with real-world data

### Climate Anomaly Alerts

Mosquitoes are expanding into new territories due to global warming. Here are the key monitoring priorities:

- **Iceland** — *Culiseta annulata* found in Oct 2025. Monitor for breeding confirmation.
- **Madeira highlands** — Warming trend may introduce new species.
- **Arctic regions** — All northern destinations: monitor for northward expansion.
- **Mediterranean** — Warming coastal areas may recruit new mosquito populations.

Report any unusual insect sightings using the `climate-anomaly` label.
