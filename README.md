## ⚠️ Critical Corrections (Read First!)

- **Seychelles is NOT mosquito-free** — it has mosquitoes, especially Dec–Feb (rainy season). Best window: May–Oct with DEET as insurance.
- **Iceland is no longer 100% mosquito-free** — *Culiseta annulata* discovered Oct 2025. Still best warm bug-free option but **verify before booking**.
- **Antarctica is now the ONLY landmass with a full mosquito-free guarantee.**
- **Patagonia = June–August ONLY** (Nov–Mar = sandfly hell, DO NOT visit if phobic).
- **Montserrat & Curaçao** are the only nearly-zero-bug Caribbean destinations.

## 📅 Seasonal Cheat Sheet

| Season | Best Destinations | Avoid |
|---|---|---|
| **Jan–Feb** | Antarctica, Madeira, Hokkaido, Atacama, Montserrat ⭐, Curaçao ⭐, UK Cities | Africa wet season, Seychelles, French Polynesia |
| **Mar–Apr** | Atacama spring, Madeira, Hokkaido end, New Caledonia shoulder, Curaçao | Seychelles, Patagonia MELTDOWN |
| **May–Jun** | New Caledonia dry season, Faroe Islands, Atacama, UAE/Oman, Montserrat ⭐, Curaçao ⭐ | Patagonia MELTDOWN, S.E. Asia |
| **Jul–Aug** | Iceland ⚠️verify, Madeira, Faroe Islands, New Caledonia, Maldives resorts, Kona Hawaii | Patagonia MELTDOWN, Africa, S.E. Asia |
| **Sep–Oct** | New Caledonia, Atacama autumn, UAE/Oman, Montserrat ⭐ peak dry, Curaçao ⭐ | Seychelles wet start, French Polynesia late wet |
| **Nov–Dec** | Atacama late, UAE/Oman, Madeira, Montserrat ⭐, Curaçao ⭐, UK Cities | French Polynesia wet, Seychelles peak |
| **Dec–Apr** | **BEST CARIBBEAN WINDOW:** Montserrat ⭐ & Curaçao ⭐ (peak dry season!) | Seychelles, most S.E. Asia |

### 🔥 Patagonia — THE SEASONAL TRAP ⚠️
- **ONLY SAFE:** June – August (their winter)
- **DO NOT VISIT:** November – March (their summer = sandfly hell)
- This is a bi-annual destination: 6 months safe, 6 months dangerous!
