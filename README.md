2. **Planning a trip?** → Browse the [Destination Quick Reference](README.md#-destination-quick-reference) above for an overview of all destinations ranked by bug safety

   - [European & S. Hemisphere Guide](guides/7-european-bug-free-spots.md) — Islands, cities, countries
4. **📬 Community updates?** → Read the [Newsletter & Updates](guides/8-newsletter.md)
5. **When to go?** → Check the [Seasonal Safety Calendar](guides/seasonal-safety-calendar.md)
6. **Want to contribute?** → Use the [Submission Template](guides/submission-template.md) **or open a GitHub Issue** using our [Issue Template](.github/ISSUE_TEMPLATE.md) to propose a new destination — then open a PR!
7. **Critical updates?** → Read our [Research Updates & Corrections](guides/research-updates.md)

> ⚠️ **Important**: Seychelles has mosquitoes (especially Dec–Feb)! It is malaria-free but NOT mosquito-free. Best visit: May–October with DEET as insurance.
